04 · How it works
How it works

Four invisible layers, one obvious experience.

Identity

zkLogin turns a Google account into a self-custodial Sui wallet. The user never sees a key.

Settlement

Balances are USDsui, a US dollar token on Sui, moved with sub-second finality.

Gas

Every send is sponsored, so the user pays nothing to transact.

Names

Everyone claims name@talise.sui, a real on-chain identity that money routes to.

Put together, sending a dollar feels like sending a text.

Send dollars without broadcasting them to the world.
